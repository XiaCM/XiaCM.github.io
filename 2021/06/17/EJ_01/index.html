Effective Java 读书笔记

Effective Java 买回来很久，却没有仔细的去研读。一来是自身懒惰，二者是大致浏览了一下，相关的编程建议，在缺乏一定的经验和踩坑经历支撑下，难以达意。
近来做项目重构，虽然从抽象的角度设计了很多接口，也写了不少针对性的工具类，但始终都是凭借长期编码的感觉和习性在做。借此契机，研读下 Effective Java ，希望能构建一些理论上的优秀意识，而不是随心而行做设计和重构。
